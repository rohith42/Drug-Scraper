Welcome to the drug API!

This API allows you to get all the drug names and some basic information about the drug. The data was webscraped from MedlinePlus.gov and more information about each drug can be found at the link provided in the get request's JSON file.

Endpoints:

  • /allnames
    • Returns a JSON file with one key, names, which is an array of all the different drug names stored in MedlinePlus.gov
  • /drug
    • Query param: name
    • Example: /drug?name=Simvastatin
    • Returns a JSON file with the specified drug's name, link, brand names, and important cautions/warnings
    • Note: The name MUST be a name from the array in /allnames and it must be spelled and capitalized as such
